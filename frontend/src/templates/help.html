Getting Started

Welcome to Transcribo! Follow these steps to get started:

1 Upload your audio file

Navigate to the Upload page and select your audio file. We support MP3, WAV, and M4A formats.

2 Choose your language

Select the primary language of your audio from the dropdown menu.

3 Start processing

Click "Start Upload" and wait for the transcription to complete.

4 View results

Once processing is complete, you can view, edit, and download your transcription.

File Upload
Supported Formats
  • MP3 - Recommended format, best balance of quality and size
  • WAV - Highest quality, but larger file size
  • M4A - Good quality, commonly used on Apple devices
Tip: For best results, ensure your audio is clear and has minimal background noise.
File Size Limits
  • Single file: up to 1GB
  • ZIP archive: up to 2GB
  • Maximum files per ZIP: 100
Batch Processing

Process multiple files at once using ZIP archives:

  1. Create a ZIP file containing your audio files
  2. Upload the ZIP file
  3. Set default language (can be overridden per file)
  4. Monitor progress for each file individually
Tip: Name your files descriptively to easily track multiple transcriptions.

Job Management
Job States
  • Pending - Job is queued for processing
  • Processing - Transcription in progress
  • Completed - Transcription finished successfully
  • Failed - Error occurred during processing
Actions
  • View - Open transcription in editor
  • Cancel - Stop processing (pending/processing jobs)
  • Download - Save transcription as text file
Settings
Processing Options
  • Quality - Balance between speed and accuracy
  • Noise Reduction - Improve quality for noisy audio
  • Speaker Detection - Identify different speakers
Display Options
  • Theme - Light, dark, or system default
  • Jobs Per Page - Number of jobs to display
  • Time Estimates - Show processing time estimates
Troubleshooting
Common Issues

Upload fails

  • Check file format and size limits
  • Ensure stable internet connection
  • Try uploading smaller files first

Poor transcription quality

  • Verify correct language selection
  • Enable noise reduction for noisy audio
  • Use higher quality audio source if possible

Processing takes too long

  • Check file size and quality settings
  • Consider splitting large files
  • Monitor system status for issues
Frequently Asked Questions

What audio formats are supported?

We support MP3, WAV, and M4A formats. MP3 is recommended for the best balance of quality and file size.

How long does processing take?

Processing time depends on file size, audio quality, and selected language. Typically, a 1-hour audio file takes 15-30 minutes to process.

Can I process multiple files at once?

Yes, you can upload multiple files in a ZIP archive. Each file will be processed individually.

How accurate is the transcription?

Accuracy depends on audio quality and language. With clear audio and correct language selection, accuracy typically ranges from 85% to 95%.

Can I edit the transcription?

Yes, completed transcriptions can be edited in our built-in editor. Changes are saved automatically.
